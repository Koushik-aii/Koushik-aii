## 🚀 Featured Projects

### 🌐 [GlobeSync](https://github.com/Koushik-aii/GlobeSync)
> Travel itinerary planning & coordination platform

A full stack travel platform supporting travelers, guides, administrators, and customer support workflows. Features trip booking, package management, payments, messaging, and support ticket handling with role-based access control.

`NestJS` `TypeScript` `MySQL` `Swagger` `REST APIs`

---

### 🔍 [CodeLens](https://github.com/Koushik-aii/CodeLens)
> AI-powered repository intelligence platform

An AI-powered tool that analyses any GitHub repository URL and generates architecture summaries, codebase explanations, project structure insights, and technology breakdowns — helping developers onboard faster into unfamiliar codebases.

`React.js` `Redux Toolkit` `Node.js` `GitHub API` `OpenAI API`

---

### 🗂 [StoriX](https://github.com/Koushik-aii/StoriX)
> Intelligent file management platform

A full stack file management system with chunked uploads, resumable transfers, metadata-driven search, access-controlled sharing, storage analytics, and file deduplication using hashing for optimized storage efficiency.

`React.js` `Redux Toolkit` `NestJS` `PostgreSQL` `Redis`
